    Database Connections

    By default, the platform is running in Local Storage Demo Mode. Any changes you make (adding members, uploading photos, linking spouses, etc.) will be stored securely inside your browser's local cache. This allows immediate previewing without any external dependencies.

    Dynamic Adapters Support

    The Rajput Family portal automatically connects to the server REST endpoints when hosted via Node.js Express & MongoDB. It also supports Firebase Firestore. Feel free to fill in configurations or host the node app to try it!
